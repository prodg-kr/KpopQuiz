# 🎤 K-pop Quiz Game

재미있는 K-pop 지식 퀴즈 게임! BTS, BLACKPINK, NewJeans 등 인기 K-pop 아티스트에 대한 당신의 지식을 테스트해보세요.

## ✨ 특징

- 📱 **반응형 디자인** - 모바일, 태블릿, 데스크톱 모두 지원
- 🎲 **랜덤 문제 순서** - 매번 다른 순서로 출제
- ♿ **접근성** - 스크린 리더 및 키보드 네비게이션 지원
- 🎨 **현대적인 UI** - 그라데이션과 애니메이션
- 🏆 **실시간 점수** - 진행률 및 점수 표시
